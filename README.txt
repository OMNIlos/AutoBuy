==============================
Telegram Gift Bot — Быстрый старт
==============================

1. Откройте терминал (или командную строку на Windows).
2. Перейдите в папку с ботом (например, через команду cd "Путь к папке").
3. Дайте права на выполнение файла start_bot.sh:
   - На Mac/Linux:   chmod 777 start_bot.sh
   - На Windows:     start_bot.bat
4. Запустите файл автозапуска:
   - На Mac/Linux:   ./start_bot.sh
   - На Windows:     start_bot.bat

Скрипт автоматически:
- создаст виртуальное окружение (venv),
- установит все необходимые библиотеки,
- запустит бота с графическим интерфейсом.

Вам не нужно ничего настраивать вручную!
